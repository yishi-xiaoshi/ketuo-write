# 可拓写作入门篇课程大纲（第1-4课）

> 面向对象：中学生（初中/高中）
> 课程目标：让学生能用可拓学的形式化工具分析写作题目，打开思路，写出有深度的文章

---

|------|------|---------|-------------|
| 第1课 | 可拓建模——看懂题目的底层语言 | 物元 / 事元 / 关系元 | 把作文题目变成可分析的结构化模型 |
| 第2课 | 发散树+相关网——打开思路的宽度与广度 | 发散分析 / 相关分析 | 从一个主题发散出多条立意路径 |

## 第1课：可拓建模——看懂题目的底层语言

### 一、课程目标

1. 理解可拓学用"基元"来形式化描述事物的思想
2. 能用物元、事元、关系元三种工具，将作文题目拆解为结构化模型
3. 建立"题目背后有结构"的意识，打破"读完题目直接动笔"的习惯

---

### 二、核心概念

#### 2.1 什么是基元？

可拓学把世界上一切"物、事、关系"用统一的形式化语言表达，这个基本单位叫**基元**。

简单说：基元 = **对象 + 特征 + 量值**（三个要素说清楚一件事）

| 要素 | 含义 | 例子 |
|------|------|------|
| 对象 | 是什么/做什么/什么关系 | 茶杯、写作、连接 |
| 特征 | 从哪些角度描述 | 颜色、目的、方式 |
| 量值 | 特征的具体取值 | 红色、抒情、嵌入 |

---

#### 2.2 物元——描述"是什么"

**定义**：用来形式化表示"物"的基本模型。一个物有很多特征，每个特征有对应量值。

**多维物元示例**（茶杯）：

```
茶杯，对象 = 茶杯
    特征：材质，量值：玻璃
    特征：颜色，量值：红色
    特征：形状，量值：圆柱体
    特征：高度，量值：10cm
    特征：容积，量值：500ml
```

**写作应用**：
- 题目给出的事物，可以用多维物元列出它的所有特征
- 每一个特征，都是一个可以写的角度

**课堂示例**：把"书"用多维物元表示

```
书，对象 = 书
    特征：载体，量值：纸
    特征：重量，量值：0.5kg
    特征：用途，量值：传递知识
    特征：作者，量值：[多种]
    特征：价格，量值：[有贵有便宜]
    特征：潜在价值，量值：改变命运
```

---

#### 2.3 事元——描述"做什么"

**定义**：用来形式化表示"事件、动作、功能、需要"的基本模型。

**事元的特征维度**（七个方面）：
- 支配对象——做什么
- 施动对象——谁做
- 接受对象——为谁做
- 时间——什么时候
- 地点——在哪里
- 方式——怎么做
- 程度——做到什么程度

**示例**：把"努力"用事元表示

```
努力，对象 = 努力
    支配对象：学习
    施动对象：学生小明
    接受对象：父母/未来的自己
    时间：高中三年
    地点：教室/家中
    方式：每日刷题+归纳错误
    程度：全力以赴
```

**写作应用**：
- 事元让你把"行为"写得具体，而不是空泛的"努力"二字
- 事元的每个特征都可以变化，变化就产生不同的故事情节

---

#### 2.4 关系元——描述"怎么联系"

**定义**：用来形式化表示两个对象之间关系的基本模型。

**示例**：把"我与奶奶的关系"用关系元表示

```
陪伴关系，对象 = 陪伴关系
    前项：我
    后项：奶奶
    程度：密切
    维系方式：每周视频通话
    时间：每个周末
```

**写作应用**：
- 文章的情感，往往来自于关系的变化
- 关系元帮你精准定位"两个对象之间发生了什么变化"

---

### 三、三种基元的对比与写作对应

| 基元类型 | 回答的问题 | 写作中的作用 |
|---------|---------|------------|
| 物元 | 这个东西是什么 | 描写景物、人物外形 |
| 事元 | 发生了什么事 | 叙事、写行动、写功能 |
| 关系元 | 两者如何联系 | 写情感关系、结构关系、因果关系 |

---

### 四、作文题目建模实战

**题目示例**："坚守"

**第一步：问这是物/事/关系？**
- "坚守"是一个动作 → 用**事元**建模

**第二步：列出事元的各特征**

```
坚守，对象 = 坚守
    支配对象：[要守住的东西：理想/底线/承诺/岗位...]
    施动对象：[谁在坚守：普通人/英雄/你自己...]
    接受对象：[为谁坚守：家人/社会/信仰...]
    时间：[多长时间：一生/关键时刻...]
    地点：[在哪里坚守：边疆/医院/课堂...]
    方式：[如何坚守：默默付出/对抗压力...]
    程度：[坚守的深度：坚如磐石/摇摆后最终坚持...]
```

**第三步：每一个特征，都是一个立意角度**

```
（坚守，支配对象，底线）→ 写诚信/道德
（坚守，支配对象，梦想）→ 写追梦/奋斗
（坚守，支配对象，岗位）→ 写责任/奉献
（坚守，施动对象，普通老人）→ 写岁月/平凡的伟大
```

---

### 五、课堂练习

**练习1**（5分钟）：选择"桥"或"光"，写出它的多维物元（至少6个特征）

**练习2**（8分钟）：选择"告别"或"传承"，写出它的事元，列出至少5个特征维度，并说出每个维度对应的写作方向

**练习3**（讨论）：你认为物元、事元、关系元中哪一种最难理解？为什么？

---

### 六、课后任务

从本周学过的一篇课文中，选择文章的核心事件，用事元形式化表示出来。

## 第2课：发散树+相关网——打开思路的宽度与广度

### 一、课程目标

1. 理解发散树方法：从一个基元出发，根据发散规则扩展出多个基元
2. 理解相关网方法：发现事物之间的相关联系，找到可以影响目标的路径
3. 用发散树+相关网，为一道作文题找出至少5条不同的立意角度

---

### 二、核心方法

#### 2.1 发散树方法

**核心思想**：任何写作对象都具有可以拓展的性质——同一个词，可以从不同角度看到不同方面；同一个角度，可以落到不同事物上；同一个事物，程度有高有低。

**发散树的五条规则**：

| 规则 | 名称 | 核心操作 | 写作意义 |
|------|------|---------|---------|
| 规则1 | 一对象多特征 | 同一主题，挖掘不同特征 | "坚守"可以是坚守底线/梦想/岗位 |
| 规则2 | 多对象一特征 | 同一特征，找不同载体 | 都具有"坚守"特质的对象有哪些 |
| 规则3 | 一特征多量值 | 同一维度，量值变化 | 坚守的"程度"可从"动摇过"到"始终如一" |
| 规则4 | 多对象同特征 | 找同类事物 | 还有哪些人体现了同样的精神 |
| 规则5 | 参变量变化 | 不同时间/条件下的变化 | 同一个人在不同阶段的坚守有何不同 |

**发散树的基本步骤**：
1. 写出要分析的基元（对象+特征+量值）
2. 选择应用发散规则
3. 从原基元扩展出多个新基元
4. 判断是否找到满意的写作角度，如未找到，继续对拓展出的基元再次发散

---

**写作实战：用发散树分析"坚守"**

```
（坚守，支配对象，梦想）
    → 发散 →
    （坚守，支配对象，底线）
    （坚守，支配对象，岗位）
    （坚守，支配对象，承诺）
    （坚守，支配对象，家园）

（坚守，施动对象，学生小明）
    → 发散 →
    （坚守，施动对象，老师）
    （坚守，施动对象，边疆战士）
    （坚守，施动对象，普通工人）
    （坚守，施动对象，非遗传人）
```

**二级发散**（对"非遗传人"继续发散）：
```
（坚守，支配对象，非遗技艺）
    → 发散 →
    （传承，方式，手把手教授）
    （困难，类型，无人学习/市场萎缩）
    （坚守者，特征，老年）
    （时间，跨度，数十年）
```

→ 立意产生：**"老人守护即将消逝的手艺"**——这就是一个有深度的写作角度！

---

#### 2.2 相关网方法

**核心思想**：客观世界中任何事物，都与其他事物存在千丝万缕的联系。如果两个事物的量值之间存在某种**函数关系**（即一个变，另一个也会跟着变），就说它们"相关"。相关网，就是找出这种"牵一发而动全身"的关系网络。

**三种相关规则**：

| 规则类型 | 形式化表达 | 通俗解释 | 写作联想 |
|---------|-----------|---------|---------|
| 同对象异特征相关 | (O, C₁, V₁) ~ (O, C₂, V₂) | 同一个人的两个特征"挂钩"了 | 一个人的"勇气"与"自信"相关 |
| 异对象同特征相关 | (O₁, C, V₁) ~ (O₂, C, V₂) | 两个不同事物在同一个维度上"挂钩"了 | "老师的付出"与"学生的成绩"都关乎"教学质量" |
| 异对象异特征相关 | (O₁, C₁, V₁) ~ (O₂, C₂, V₂) | 两个不同事物的不同特征"挂钩"了 | "一个人的坚持"会影响"整个团队的士气" |

**相关网的基本步骤**：
1. 写出要分析的基元（对象+特征+量值）
2. 根据**领域知识或常识**，找出与该基元量值存在函数关系的其他基元
3. 画出相关网络，明确"谁变，谁跟着变"
4. 从相关网中选取节点，作为写作切入点或情节推进的依据

---

**写作实战：用相关网分析"坚持"**

**背景**：一个人坚持晨跑

**步骤1：列出基元**
```
（小明，坚持晨跑，是）
```

**步骤2：找相关关系**
```
（小明，坚持晨跑，是）
    ↓ 同对象异特征相关
（小明，身体素质，提升）    ← 晨跑久了，体质变好
    ↓ 同对象异特征相关
（小明，学习精力充沛，程度提升）    ← 体质好，上课不犯困
    ↓ 异对象同特征相关
（成绩，排名，提升）    ← 精力好 → 成绩提升
```

**步骤3：继续扩展相关网**
```
（小明，坚持晨跑，是）
    ↓ 同对象异特征相关
（小明，作息规律，是）    ← 晨跑倒逼早睡
    ↓ 异对象同特征相关
（父母，心情，安心）    ← 看到孩子自律，家长放心
    ↓ 异对象异特征相关
（家庭氛围，和睦）    ← 家长少操心，家庭关系改善
```

→ 相关网揭示了：**"一个人的晨跑，牵动了整个家庭的幸福"**

→ 立意：**"你以为在坚持自己，其实你也在治愈家人"**

---

### 三、发散树 vs 相关网的区别

| 工具 | 核心操作 | 适合找什么 |
|------|---------|----------|
| 发散树 | 从一个基元向外"发散" | 更多角度、更多事物、更多特征 |
| 相关网 | 分析对象之间的"联系" | 立意的因果逻辑、事物的传导效应 |

> 发散树解决"写什么"，相关网解决"为什么这样写"。

---

### 四、课堂综合练习

**题目**："距离"

**步骤一**（发散树，8分钟）：
1. 给"距离"建物元/事元模型
2. 用发散规则，找出至少6个不同立意角度

**步骤二**（相关网，7分钟）：
1. 选择其中一个立意角度
2. 用相关网分析"这个距离的存在/消除，会影响什么"
3. 从影响关系中提炼一句完整的立意

**步骤三**（分享，5分钟）：小组代表分享，全班投票最有深度的立意

---

### 五、课后任务

用发散树方法，对"平凡"这个词找出不少于8个写作角度，并用一句话说出每个角度的立意。

## 第3课：蕴含系+共轭部——写出深度与立体感

### 一、课程目标

1. 理解蕴含系方法：挖掘"因为…所以…""只有…才能…"的因果逻辑链
2. 理解共轭部方法：从物质/系统/动态/对立四个维度全面分析写作对象
3. 能用蕴含系找到立意的逻辑支撑，用共轭部分析人物/事物的多面性

---

### 二、核心方法

#### 2.1 蕴含系方法

**核心思想**：世界上许多事物之间存在"蕴含关系"——一件事成立，另一件事必然跟着成立。因果关系是最常见的蕴含关系。"因为"链挖得越深，文章的立意就越有说服力。

**两种蕴含关系**：

| 类型 | 说明 | 写作应用 |
|------|------|---------|
| 因果蕴含 | B₁实现，B₂必然跟着实现 | "实现梦想"→"需要坚持"→"需要克服惰性" |
| 存在蕴含 | B₁存在，B₂必然跟着存在 | "有灯光"→"有灯座"→"有电路" |

**两种组合形式**：
- **与蕴含**（两个条件缺一不可）：做成一件事需要同时满足A和B
- **或蕴含**（满足其一即可）：做成这件事，A能实现或B也能实现

**蕴含系的基本步骤**：
1. 列出要分析的写作目标
2. 向上追溯："要实现这个目标，需要什么前提？"（因果链上游）
3. 向下追溯："实现了这个目标，会带来什么结果？"（因果链下游）
4. 在链条中寻找最有价值的写作切入点

---

**写作实战：用蕴含系分析"提高学习成绩"**

```
提高学习成绩（写作目标）
  ↑
  ├── 提高理解能力
  │     ↑
  │     ├── 多读书，建立知识体系
  │     └── 主动提问，打通知识盲区
  │
  └── 提高学习效率
        ↑
        ├── 优化时间管理
        └── 减少干扰（手机/社交媒体）
```

**运用蕴含系找立意**：
- 不要只写"努力就能成功"（太浅）
- 追溯蕴含链：成功 ← 坚持 ← **克服内心的恐惧与懈怠** ← 认清自我
- 立意升华：**"真正的努力，是在无人知晓时与自己的弱点抗争"**

---

**蕴含系让文章有纵深感**

| 层次 | 写法 | 效果 |
|------|------|------|
| 只写表层 | "我努力了，所以成功了" | 平淡，无说服力 |
| 加入蕴含链 | "我能成功，是因为克服了恐惧，恐惧来自不自信，不自信来自从未被认可…" | 有纵深，真实感强 |

---

#### 2.2 共轭部方法

**核心思想**：看一个事物，不仅要看它"是什么"，还要从四个维度全面认识它：物质上由什么构成？它处在什么系统里？它现在是什么状态？它有什么对立面？四个维度都看清楚了，文章才能写得立体。

**四对共轭部**：

| 维度 | 共轭对 | 说明 | 写作意义 |
|------|-------|------|---------|
| 物质性 | 虚部 ↔ 实部 | 实部=物质实体；虚部=精神/形象/情感 | 写人物既写外在行为(实)又写内心世界(虚) |
| 系统性 | 硬部 ↔ 软部 | 硬部=组成部分；软部=内外关系 | 写事件要写"关系网"，不只写孤立的人 |
| 动态性 | 潜部 ↔ 显部 | 显部=已显现的；潜部=潜在的、未来的 | 写成长要写"潜力被激发"的过程 |
| 对立性 | 负部 ↔ 正部 | 关于某特征的正面来源和负面来源 | 写人物的优缺点时，二者共存于同一个人 |

---

**写作实战：用共轭部分析人物"老师"**

| 共轭对 | 内容 |
|-------|------|
| 实部（物质） | 粉笔、讲台、教科书、板书、外形 |
| 虚部（非物质） | 职业精神、对学生的期望、教育理念、影响力 |
| 硬部（组成） | 知识储备、备课能力、表达能力 |
| 软部（关系） | 与学生的情感纽带、与家长的沟通、与同事的合作 |
| 显部（已显现） | 当前的教学成果、可见的付出 |
| 潜部（潜在） | 某个学生多年后回忆起的那句话、播下的种子 |
| 正部（正向） | 启发智慧、传递温暖、激励成长 |
| 负部（负向） | 可能的严苛、某些不完美、职业倦怠 |

**立意来自共轭部**：
- 只写实部 → 太浅（外貌描写+上课情节）
- 实部+虚部 → 有深度（外表的疲惫 vs 内心的热忱）
- 潜部的蕴含 → 最动人（"那句被遗忘的鼓励，在15年后改变了一个人的选择"）

---

### 三、蕴含系 vs 共轭部的互补

| 工具 | 方向 | 解决的写作问题 |
|------|------|--------------|
| 蕴含系 | 纵向（因果逻辑链） | 文章立意要有说服力，论据要有支撑 |
| 共轭部 | 横向（全面结构分析） | 人物/事物要写得立体，不能单一 |

> 好文章 = 蕴含系给的**逻辑纵深** + 共轭部给的**结构立体感**

---

### 四、课堂综合练习

**题目**："那一刻，我读懂了 ______"

**步骤一**（共轭部，8分钟）：
1. 选一个填空对象（如：父亲、责任、失败）
2. 用四对共轭部对它进行全面分析，填写共轭部分析表

**步骤二**（蕴含系，7分钟）：
1. 从共轭部中选最打动你的那个角度
2. 向下追溯蕴含链："读懂这一刻"是因为什么→因为什么→因为什么
3. 找到最深层的原因，作为文章的立意核心

**步骤三**（写一段，10分钟）：
1. 用刚才的分析，写一段200字以内的开篇
2. 要求：同时体现"虚实共轭"和"蕴含逻辑"

## 第4课：分合链+变换法——组织材料+找到新立意

### 一、课程目标

1. 理解分合链方法：通过分解、组合两种操作，重新处理写作素材
2. 理解基本变换方法（置换/增删/扩缩/复制/传导）：用形式化工具改造写作对象和角度
3. 能用分合链+变换法，把普通素材变成独特的写作立意

---

### 二、核心方法

#### 2.1 分合链方法

**核心思想**：写作素材不是死的——可以拆开来看，也可以和其他素材拼起来。分合链就是"怎么拆、怎么合"的工具。

**两种可扩操作**：

| 操作 | 说明 | 写作应用 |
|------|------|---------|
| **分解** | 把一个整体拆成多个部分 | 把笼统的概念拆成几个具体的切入点 |
| **组合** | 把多个部分拼成新整体 | 把两个不相关的素材组合在同一篇文章里 |

**分合链的基本步骤**：
1. 将要分析的对象用基元表示（对象是什么，它的特征是什么）
2. 用发散树对基元进行拓展，找出多个相关基元
3. 判断这些基元之间是否可以：拆开（分解）、拼在一起（组合）
4. 考察处理后的基元是否产生了新的写作价值

---

**写作实战：用分合链处理素材**

**原素材**：奶奶每天早上5点起床做早饭

**直接写**：平淡，只是一件家常小事

**用分合链处理**：

*① 分解：把"做早饭"拆成多个步骤*
```
起床 → 选食材 → 洗菜切菜 → 生火煮饭 → 盛碗放桌 → 等人起来
```
→ 每一个步骤都可以细写，发现爱的细节

*② 组合：把"奶奶的早饭"与"我第一次为奶奶做早饭"放在一起*
```
奶奶为我做早饭 + 我为奶奶做早饭
```
→ 结构对比：**时间流转，角色互换，那一刻"长大"有了具体的形状**

---

#### 2.2 基本变换方法

**核心思想**：创新常常来自"换个角度看问题"。变换就是主动改变基元中的某个要素——换对象、增减内容、变大小、复制一个、传导连锁反应。不同的变换方式，产生完全不同的立意。

**五种基本变换**（写作重点）：

| 变换名称 | 操作说明 | 写作意义 | 典型立意 |
|---------|---------|---------|---------|
| 置换变换 | 把基元中的某个要素换成另一个 | 换主角、换视角、换时代背景 | 换个身份看问题 |
| 增删变换 | 在基元中增加或删除内容 | 给素材增加新维度，或删去冗余枝节 | 增减材料、详略得当 |
| 扩缩变换 | 把量级扩大或缩小 | 拉长时间线，或聚焦单一细节 | **以小见大** |
| 复制变换 | 复制一个或多个相同基元 | 强调数量、叠加效果 | 积少成多、群体画像 |
| 传导变换 | 改变一个基元后，引发相关基元连锁改变 | 描写一个变化如何引起连锁反应 | 连锁反应、一石激起千层浪 |

---

**扩缩变换·以小见大案例**：

**原素材**：中考前，妈妈每天给我送宵夜

**直接写**：妈妈很关心我 → 过于平淡

**用"以小见大"处理——缩小视角，放大意义**：
```
一碗宵夜
    ↓
一个小动作（轻轻关门、调低电视音量、把空调调高一度）
    ↓
一个沉默的习惯（坚持了300多天）
    ↓
→ 立意升华：**"妈妈从不夸我，但她做的每一件小事，都在替她说那句话"**
```

**扩缩变换·由小见大案例**：

**原素材**：老家门前有一棵柿子树

**用"以小见大"处理——小事切入，大义升华**：
```
柿子树 → 柿子红了 → 掉落一地 → 无人采摘
    ↓
"那棵树，就像爷爷——曾经撑起一个家，如今安静地等在那里"
    ↓
→ 立意：**"有些老去，不是凋零，是把位置让给下一代的柿子"**
```

---

**五种变换完整实战**：

**原素材**：同学借了我的书，没有还

| 变换 | 操作 | 立意效果 |
|------|------|---------|
| 置换 | 把"同学"置换为"已故的父亲" | "那本再也要不回来的书，是我最后一次与他的连接" |
| 增删 | 增加"时间维度" | 两个时间轴交叉写——现在发现书/回忆当年的交往 |
| 扩缩 | 把"一本书"扩大到"所有借出去的东西" | "有些借出去的，是再也要不回来的年华" |
| 复制 | 复制"借书"为"借了10次书" | "每次借书都是一个小借口，其实只是想和他多说两句话" |
| 传导 | 从"没还书"追溯连锁反应 | "因为这本书，我发现了他的秘密；因为这个秘密，我们成为挚友" |

### 三、分合链 + 变换法的组合使用

```
原始素材
    ↓
分合链处理（分解/组合）
    ↓
发现新的基元关系
    ↓
变换法改造（置换/增删/扩缩/复制/传导）
    ↓
独特立意诞生
```

### 四、四节课工具总览

```
第1课：可拓建模
    物元（描述事物）→ 事元（描述行为）→ 关系元（描述关系）
    作用：把模糊的题目变成可分析的结构

第2课：发散树 + 相关网
    发散树（向外扩展，找角度）→ 相关网（分析相关，找传导路径）
    作用：打开思路宽度，找到多条立意路径

第3课：蕴含系 + 共轭部
    蕴含系（纵向因果链）→ 共轭部（横向全面分析）
    作用：给立意增加深度和立体感

第4课：分合链 + 变换法
    分合链（分解/组合）→ 变换法（置换/增删/扩缩/复制/传导）
    作用：重组素材，创造独特角度
```

### 五、课堂综合练习（完整写作流程体验）

**题目**："窗外"

**Step 1 建模**（5分钟）：给"窗外"建物元，列出至少6个特征

**Step 2 发散**（5分钟）：用发散树，从物元的某个特征出发，发散出5条立意

**Step 3 挖深**（5分钟）：选一条立意，用蕴含系追溯至少3层"为什么这样立意"

**Step 4 变形**（5分钟）：用变换法（任选一种）改造这个立意，看是否产生新的角度

**Step 5 定稿立意**（2分钟）：根据以上分析，写出一句完整的、有深度的立意
